## C++打造俄罗斯方块的教程
	
如何使用C++从零创建一个俄罗斯方块游戏。

### 环境安装及游戏设计
ubuntu 18.04
仅仅依赖于SDL框架。
```sh
sudo apt-get install libsdl1.2-dev libsdl-gfx1.2-dev
```

这个游戏支持以下几种操作

| 键盘按键  |    操作说明        |
|---|------------|
|ESC|	离开游戏          |
|z	 |旋转方块
|x	 |方块快速下降         |
|Left,Right, Down |	不解析的操作 |
